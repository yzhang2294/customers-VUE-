# customers

> 使用 Vue 框架搭建的用户管理系统。

```bash
 1.使用 json-server，连接本地接口读取 json 格式的用户数据。
2.可以对数据进行增删查改。
3.不同的操作界面封装成不同 vue 组件，通过 vue-router 进行跳转
4.界面之间可以进行通信：如删除用户后会跳转到主页并显示\*\*用户已被删除
```
